Configuring Certbot for NGINX

3 Dec 2025

Words: 148

Reading Time: 1 mins

Background

I have a cloud server that runs NGINX to serve my web apps including this blog. To serve the site securely, I used certbot to issue a TLS certificate for my domain. This post captures the steps and learning.

Glossary

TLS - A TLS certificate (SSL certificate) is a file that secures internet connections by encrypting data between the browser and website server. NGINX - A high performance web server certbot - An open-source tool to automate the process of obtaining and renewing free TLS certificates from Let’s Encrypt Let’s Encrypt - Let’s Encrypt is a Certificate Authority that provides free TLS certificates

Why

Let’s Encrypt certificates are valid only for 3 months. Hence, it is essential to setup an automated mechanism to renew the certificate in order to keep the site secure and accessible. The webroot method allows certbot to renew certificates without stopping nginx.
